# CEDIM Web Portal

Sitio web oficial del **Centro de Desarrollo e Investigación en Mecatrónica (CEDIM)** de la Universidad Nacional de Ingeniería (UNI).

Este proyecto tiene como objetivo centralizar y difundir:

- Convocatorias de concursos y oportunidades académicas
- Información de talleres y actividades internas
- Proyectos en desarrollo (cronogramas, avances y documentación)
- Sistema de inventario del centro
- Gestión de recursos y documentación técnica
